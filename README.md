# mitoribosomalProteins
Repository of scripts used for a study: A pan-eukaryotic analysis of mitochondrial ribosomes uncovers major evolutionary transitions


During the preparation of this work, the authors used OpenAI ChatGPT vesrions 5 to 5.5 from August 2025 to May 2026 to generate and review scripts used for data analysis. After using this tool, the authors reviewed and edited the content as needed and takes full responsibility for the content of the published article.
